核心能力

常用 SVN 操作，集中在一个清晰界面。

不隐藏 Subversion 的工作方式，而是用更符合 macOS 的方式呈现它。

⌁

清晰掌握本地改动

以平铺或语义分组方式查看 modified、added、deleted、missing、conflicted、ignored 等状态，并直接执行 Add、Delete、Move、Copy 和 Revert。

±

文本 Diff 与安全回退

文本文件显示真实 patch；二进制文件提供安全预览与 BASE 保存能力。

↑

可控的提交流程

选择提交范围、校验消息、最终确认、查看实时输出，并在失败后保留输入。

◷

Revision History

按 revision、作者、日期和 changed paths 浏览记录，并比较两个 revision。

▦

Repository Browser

浏览本地工作副本和远程仓库，支持 revision、历史、预览、Import、Export，以及 repository-side New Folder / Copy / Move / Delete。

原生体验

复杂操作，保持透明和可恢复。

每个长时间 SVN 操作都提供进度、取消、结果和诊断信息。优先解析 XML 输出，敏感凭据不会进入日志。
